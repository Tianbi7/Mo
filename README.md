# AirCalligraphy 空中水墨书法交互系统 README
## 项目简介
AirCalligraphy 是一套**仅基于 MediaPipe Hand 手部关键点检测 + OpenCV 2D像素图像处理**的轻量化裸手国风书法交互系统，全程不使用任何3D建模、虚拟场景、游戏渲染引擎，依靠普通摄像头即可实现无介质空中水墨创作。
项目跳出传统手势绘画工具的定位，以东方书法笔墨气韵、传统书写哲学为核心，融合多模态自然裸手交互，将传统书法艺术转化为可实时参与、沉浸式体验、轻量化落地的数字交互装置，面向 SIGGRAPH Asia Poster 图形学、数字文化艺术赛道完成开发。

### 核心立意
以极简2D机器视觉算力复刻千年笔墨气韵，摒弃三维渲染的高算力桎梏；依托人本直觉的裸手交互消解数字工具冰冷感，打造集创作、美育、文化叙事于一体的东方水墨数字化传承范式，适用于数字文博、国风美育课堂、公共沉浸式艺术展陈等场景。

## 技术栈
- 计算机视觉：OpenCV-Python（像素融合、形态学水墨仿真、轮廓矫正、仿射变换）
- 手部感知：MediaPipe Tasks HandLandmarker 高精度手部关键点检测
- 数值计算：NumPy
- 运行环境：纯CPU运行，无需独立GPU，普通笔记本实时流畅运行
- 模型：本地加载 hand_landmarker.task 离线手部检测模型，无需联网

## 项目特性 & 完整交互功能
### 一、基础水墨创作交互
1. 单食指空中滑动书写，指尖移动速度动态控制笔画粗细，模拟毛笔顿笔、行笔质感
2. 手掌朝向自动切换书法风格
   - 掌心朝外（朝向镜头）：楷书，笔画锐利干净
   - 掌心朝内（背对镜头）：行书，自动生成水墨晕染扩散效果
3. 拇指+食指捏合循环切换墨色：黑墨、朱砂红墨、浅灰墨三色

### 二、画布编辑交互
1. 五指完全张开保持0.3s：一键清空整张水墨画布，重置缩放平移参数
2. 双手同时伸出，双食指联动：
   - 双指开合距离变化：整体放大/缩小书法图层
   - 双手整体移动：平移全部墨迹内容
3. 五指并拢静止3秒：轻量化轮廓算法自动矫正手写汉字，消除空中书写抖动歪斜

### 三、自动化留存交互
画布静置无操作5秒，自动将完整水墨作品保存至本地 `output_calligraphy` 文件夹，自动编号归档，方便作品导出展示。

### 四、视觉优化底层机制
1. 关键点时序滑动均值滤波，大幅抑制手部抖动，减少笔画断线
2. 纯2D仿射变换实现图层缩放平移，无任何三维空间运算
3. 实景摄像头画面与水墨图层像素融合，实景叠加墨迹，无独立虚拟白板

## 环境部署
### 1. 依赖安装
```bash
pip install opencv-python mediapipe numpy
```
### 2. 文件准备
1. 将 MediaPipe 官方 `hand_landmarker.task` 模型文件放置于代码同目录；
2. 程序运行时自动生成 `output_calligraphy` 文件夹，用于存储生成的书法作品。

### 3. 运行启动
```bash
python air_calligraphy.py
```
### 4. 按键操作
- `Q`：退出程序

## 项目文件结构
```
AirCalligraphy/
├── air_calligraphy.py        # 主程序完整运行代码
├── hand_landmarker.task      # MediaPipe离线手部检测模型
└── output_calligraphy/       # 自动生成，存储书法作品截图
```

## 运行说明
1. 程序启动自动调用本机摄像头，画面为镜像模式，符合自然书写视角；
2. 画面分为两个窗口：
   - Air Calligraphy：实景摄像头画面 + 叠加水墨笔迹，主交互窗口
   - Ink Canvas：纯水墨图层预览，仅显示笔迹，便于观察水墨渲染效果
3. 界面左上角实时提示当前书法风格、墨色、手部检测状态与全部手势操作说明。

## 创新亮点（适配SIGGRAPH Asia Poster投稿）
1. **零建模、零3D场景轻量化方案**：全部效果依托OpenCV像素矩阵运算实现，规避传统数字艺术项目高成本三维渲染管线；
2. **物理级水墨仿真逻辑**：基于指尖运动时序、速度驱动笔画粗细与墨晕扩散，还原传统毛笔书写韵律；
3. **分层多模态手势判定体系**：单手书写、双手变换、捏合换色、手掌清空等手势互不冲突，上手门槛极低；
4. 文化与技术双向价值：兼顾实时图形学轻量化创新、传统书法数字化传承、公共艺术场景落地能力三大核心价值；
5. 低算力高实时性：普通CPU即可稳定25–35FPS实时运行，现场演示部署便捷，可复现性强。

## 拓展优化方向（后续迭代高阶文化交互）

3. 手势调取历代碑帖拓印参考图层，实现线上临摹教学；
4. 双人协同创作，增加红色篆刻印章手势交互，复刻传统文人书画唱和形式；
5. 岁月风化滤镜，手势滑动控制宣纸老化、古帖裂纹质感特效。
